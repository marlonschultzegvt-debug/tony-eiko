Validacao do Tony & Eiko App

Arquivo simples para conferir o MVP antes de qualquer publicacao externa.

Como abrir

Com o servidor rodando, acesse http://localhost:5190/.

Comandos:

npm install

npm run dev -- --port 5190

Checklist de validacao

  1. Escolher os perfis Cliente, Profissional e Gestao na tela de acesso.
  2. Abrir a home e conferir logo, foto principal e botoes.
  3. Criar um agendamento com servico, profissional, data e horario.
  4. Confirmar se o agendamento aparece como pendente no painel.
  5. Testar botoes Aceitar, Concluir e Recusar.
  6. Abrir Indicadores e conferir se os numeros mudam.
  7. Abrir Config, alterar dados do salao e exportar o backup JSON.
  8. Abrir Implantacao e conferir o roadmap de producao.
  9. Conferir se o navegador reconhece o manifesto PWA.
  10. Conferir no celular se textos e botoes nao ficam cortados.

Observacao

Esta versao ainda usa dados locais no navegador. Para operacao real, precisa banco de dados, login e integracao de notificacoes.
